Provision Uguisu & Guillemot

1. Generate secret

Create a new AES-128 key. Do this once per pair of devices.

2. Flash Key Fob (Uguisu)

Plug the fob in via USB-C, then click below. Select the serial port when prompted.

3. Flash Receiver (Guillemot)

Plug the receiver in via USB-C, then click below. Use the same key as above.

Done

Unplug both devices. They are now paired and ready for deployment.
